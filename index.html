O que é AngularJS?

Framework mantido pela Google, utilizado para criar aplicações Single Page. Além disso, permite ao programador criar suas próprias tags e atributos HTML.

As SPAs são aplicações completas, desenvolvidas em JavaScript, que funcionam quase como se estivessem sendo executadas nativamente no desktop. Não tem reload

Características do AngularJS

Modularização

- Estrutura de diretorios, como é organizado
- Sub-modulos e dependências
- Reuso de modulos em varas partes do projetos
- MVWE angular, onde fica o que

Controllers

```js angular.module('app.principal',[]) .controller('MainController',['$scope',function($scope){ //Código aqui }]) ``` ```html <div ng-controller="MainController"> <!-- Html aqui--> </div> ```

- Controller trabalha como um gerenciador de dados entre a view e a service
- Guarda a logica de apresentação dos dados na view
- Trata da interações da view

Services

```js angular.module('app.principal',[]) .service('MainService',['$http',function($http){ //Código aqui }]) ```

- Responsável pelas requisições com APIs
- Envia chamadas
- Retorna dados

Injeção de Dependência

Modo com que o Angular faz sua modularização.

```js angular.module('app.principal', ['app.marketing', 'app.financeiro', 'app.desenvolvimento' ]) ```

- Possibilita o reuso de serviços entre módulos
