MongoDB

MongoDB é uma aplicação de código aberto, de alta performance, sem esquemas, orientado a documentos. Foi escrito na linguagem de programação C++. Além de orientado a documentos, é formado por um conjunto de documentos JSON.

Comandos Gerais

- use : é usado para criar um banco, caso ele não exista
- show dbs : imprime a lista de todos os bancos no server
- show collections: imprime a lista de todos as collections no atual banco
- coll = db. : atribui uma collection do banco a uma variável para ser usado para pesquisas

Comando find()

- db.collection.find(query, projection) ou coll.find()
- query: objeto que define criterios de seleção para o find usando queryOperators
Exemplo: {field1:{$gt:12}}
- Projection: objeto que especifica os campos que serão retornados no find
Exemplos: {field1:true,field2:false}

Comandos insert(), update(), save() e remove()

- coll.insert(): insere um ou varios documentos em determinada collection
- coll.update(): atualiza um ou muitos documentos de uma determinada collection
- coll.save(): atualizar ou insere um novo documento, dependendo dos paramentros do documento
- coll.remove(): remove documentos da coleção dada o parametro "query" no mesmo formato do find()

O que é AngularJS?

Framework mantido pela Google, utilizado para criar aplicações Single Page. Além disso, permite ao programador criar suas próprias tags e atributos HTML.

As SPAs são aplicações completas, desenvolvidas em JavaScript, que funcionam quase como se estivessem sendo executadas nativamente no desktop. Não tem reload

Características do AngularJS

Modularização

- Estrutura de diretorios, como é organizado
- Sub-modulos e dependências
- Reuso de modulos em varas partes do projetos
- MVWE angular, onde fica o que

Controllers

```js angular.module('app.principal',[]) .controller('MainController',['$scope',function($scope){ //Código aqui }]) ``` ```html <div ng-controller="MainController"> <!-- Html aqui--> </div> ```

- Controller trabalha como um gerenciador de dados entre a view e a service
- Guarda a logica de apresentação dos dados na view
- Trata da interações da view

Services

```js angular.module('app.principal',[]) .service('MainService',['$http',function($http){ //Código aqui }]) ```

- Responsável pelas requisições com APIs
- Envia chamadas
- Retorna dados
